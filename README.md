# Site Redirector Pro 🛑

网站重定向助手 - 拦截分心网站，冷静倒计时后重定向到指定网站，帮助你保持专注。

## 功能特点

- **冷静期机制**：30秒倒计时，给你时间思考是否真的要摸鱼
- **红蓝药丸选择**：冷静期结束后，选择「回去干活」或「继续摸鱼」
- **临时绕过**：选择摸鱼后 5 分钟内不再拦截，给你喘息空间
- **拦截计数**：显示今日次数和累计次数，直面自己的摸鱼频率
- **随机语录**：每次拦截显示一言语录，激励或警醒
- **随机标题**：灵魂拷问 + 温和提醒，每次不同的文案
- **主题切换**：支持明亮/暗黑模式，可跟随系统或手动设置
- **每日配额**：为黑名单站点设置每日可访问分钟数/次数
- **本周趋势**：查看近 7 天拦截走势与高峰时段
- **站点排行**：统计被拦截最多的网站排名
- **黑名单管理**：通过菜单随时添加、删除、编辑拦截网站
- **自定义目标**：可设置重定向到任意网站

## 安装

### 前置要求

请先安装以下任一用户脚本管理器：
- [Tampermonkey](https://www.tampermonkey.net/) (推荐)
- [Violentmonkey](https://violentmonkey.github.io/)
- [Greasemonkey](https://www.greasespot.net/)

### 安装脚本

**方式一：从 Greasy Fork 安装**

[![Install](https://img.shields.io/badge/Install-Greasy%20Fork-red)](https://greasyfork.org/scripts/563551)

**方式二：手动安装**

1. 点击 [site-redirector.user.js](./site-redirector.user.js)
2. 点击 "Raw" 按钮
3. 用户脚本管理器会自动弹出安装提示

## 使用

脚本安装后自动生效。访问被拦截的网站时：

1. 页面显示拦截提示和 30 秒倒计时
2. 冷静期内可点击「算了，回去干活」立即跳转
3. 倒计时结束后出现红蓝药丸选择：
   - 💼 **蓝色药丸**：回去干活，跳转到工作页面
   - 🎮 **红色药丸**：继续摸鱼，5 分钟内不再拦截

## 菜单命令

点击浏览器中的 Tampermonkey 图标，可使用以下功能：

| 命令 | 说明 |
|------|------|
| 🎯 设置重定向目标 | 自定义跳转到哪个网站 |
| 📋 查看黑名单 | 查看当前拦截的网站列表 |
| ➕ 添加网站到黑名单 | 添加新的拦截网站 |
| ➖ 从黑名单移除网站 | 移除不想拦截的网站 |
| ✏️ 编辑完整黑名单 | 批量编辑所有拦截网站 |
| 🔙 重置为默认黑名单 | 恢复默认拦截列表 |
| 🔄 重置拦截计数 | 清零拦截统计 |
| 📊 查看拦截统计 | 查看今日/累计拦截次数等信息 |
| ⏱️ 设置每日配额 | 设置每日可访问分钟数/次数 |
| 📈 查看本周趋势 | 查看近 7 天拦截趋势与高峰时段 |
| 🏆 查看站点排行 | 查看被拦截最多的网站排行 |
| 🎨 切换主题模式 | 切换明亮/暗黑/跟随系统 |

## 默认配置

- **默认黑名单**：bilibili.com, douyin.com, weibo.com, x.com
- **默认重定向目标**：https://claude.ai
- **冷静期时长**：30 秒
- **临时绕过时长**：5 分钟
